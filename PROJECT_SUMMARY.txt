================================================================================
                            INSTALLATION QUICK START
================================================================================

SERVER (Paper 1.21.11+):
  1. Copy server-paper-0.1.0-SNAPSHOT.jar to plugins/
  2. Start server (generates config)
  3. Edit plugins/Evilkaraoke/config.yml with your API endpoint
  4. Run: /ek reload

CLIENT (Minecraft 1.21.11):
  Fabric:   Copy client-fabric-0.1.0-SNAPSHOT.jar to mods/
  NeoForge: Copy client-neoforge-0.1.0-SNAPSHOT.jar to mods/

FIRST USE:
  /ek help                    # Show commands
  /ek request <song name>     # Queue a song
  /ek current                 # What's playing?
  /ek pause / resume / skip   # Playback controls (ops)

================================================================================
                               KEY COMMANDS
================================================================================

For All Players:
  /ek request <query>    - Queue a song
  /ek search <query>     - Search without queueing
  /ek queue              - Show upcoming tracks
  /ek current            - Show now playing
  /ek randomsong         - Queue random song
  /ek stats me           - Your statistics

For Operators:
  /ek pause / resume     - Playback control
  /ek skip / stop        - Skip or stop
  /ek audience <target>  - Set who hears audio (@a/@s/player)
  /ek radio <station>    - Start radio mode
  /ek doctor             - Diagnostics
  /ek listeners          - Connected clients

================================================================================
                              ARCHITECTURE
================================================================================

Paper Server:
  - /evilkaraoke commands → EvilkaraokeCommand.java
  - Queue management → KaraokeSession.java
  - Playback coordination → PlaybackCoordinator.java
  - API integration → NeurokaraokeClient.java
  - Statistics → StatsService.java

Client Mods (Fabric + NeoForge):
  - Audio playback → JavaSoundAudioBackend.java
  - Spatial volume → PlaybackGain.java
  - Network → custom payload channels (JSON)
  - Zero game logic or commands

Protocol:
  - evilkaraoke:hello (C→S) - Client handshake
  - evilkaraoke:audio (S→C) - Audio commands (PLAY/PAUSE/STOP/etc)
  - evilkaraoke:status (C→S) - Client playback state
